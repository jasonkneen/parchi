Privacy Policy

Parchi stores settings, BYOK API keys, and local chat/session state in your browser using chrome.storage.local.

In BYOK mode (bring your own key), requests go directly from your browser to the configured AI provider.

In managed billing modes (credits or provisioned OpenRouter keys), requests and billing events use our Convex backend. We store only the account and billing data needed for entitlement checks and payment reconciliation.

For OpenRouter key provisioning, we store key metadata (for example key hash, selected model, and optional guardrail id). We do not intentionally store the OpenRouter key secret after issuing it.

We use Stripe for payment processing and do not store raw credit card details.
